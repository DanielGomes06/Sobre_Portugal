# 🇵🇹 Site Portugal

Este é um site informativo sobre Portugal, desenvolvido com foco em apresentar aspectos culturais, turísticos, históricos e gastronômicos do país.

## 📌 Funcionalidades

* Página inicial com destaque visual (banner com carousel)
* Navegação responsiva com menu adaptado para dispositivos móveis
* Seções dedicadas:

  * Culinária
  * Cultura
  * Turismo
  * História
* Layout em cards para facilitar a navegação
* Tabela com informações gerais sobre Portugal

---

## 💻 Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **Bootstrap 5.3**

O Bootstrap foi utilizado para:

* Criar um layout responsivo (grid system)
* Estilizar componentes como:

  * Navbar
  * Cards
  * Botões
  * Tabelas
* Implementar o menu mobile (navbar colapsável)
* Utilizar o carousel para o banner principal

---

## 📱 Responsividade

O site foi desenvolvido com foco em responsividade, utilizando o Bootstrap para adaptação em diferentes tamanhos de tela:

* Desktop
* Tablet
* Dispositivos móveis

A navbar se transforma em menu “hambúrguer” em telas menores.
