 📖 Sobre o projeto

Este site foi desenvolvido com o objetivo de apresentar informações sobre Portugal de forma simples, organizada e visualmente agradável. O projeto reúne conteúdos sobre cultura, história, turismo e curiosidades do país, proporcionando uma navegação intuitiva para o usuário.

Para o desenvolvimento da interface, foi utilizado o framework **Bootstrap**, o que permitiu a criação de um layout responsivo e adaptado para diferentes dispositivos, como computadores, tablets e celulares.
